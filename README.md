# GitHub YouTrack Time Tracker

Add time to YouTrack issues directly from GitHub PRs.

## Install

**Firefox:** Download [.xpi from Releases](https://github.com/qol-tools/github-youtrack-time/releases)
**Chrome/Edge:** Download Chrome .zip, extract, load unpacked

## Usage

1. Open any GitHub PR with a YouTrack ID in the title (e.g., `ABC-123`)
2. Click "Add Time" button
3. First time: enter subdomain, token, work type (saved locally)
4. Type time: `5m`, `2h`, `1d`
