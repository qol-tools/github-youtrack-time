# GitHub YouTrack Time Tracker

Add time tracking to YouTrack issues directly from GitHub Pull Requests.

## Installation

### Firefox

Download the latest `.xpi` from [Releases](https://github.com/qol-tools/github-youtrack-time/releases)

### Chrome/Edge

Download the latest Chrome `.zip` from [Releases](https://github.com/qol-tools/github-youtrack-time/releases), extract it, and load unpacked.

## Setup

1. Open a GitHub PR with a YouTrack issue ID in the title (e.g., `ABC-123`)
2. Click "Add Time" button
3. Enter your YouTrack subdomain
4. Enter your YouTrack token
5. Select work item type
6. Done

## Usage

Type time like: `5m`, `2h`, `1d`
